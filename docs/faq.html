FAQ

How does it work?

First, you connect to your wallet. THen you upload an image, name, and description. The image will be pinned to IPFS and a metadata file with the name, description, and reference to the image will also be pinned. The final step is to mint the NFT by calling the smart contract and passing in the location of that metadata file.

What is IPFS?

IPFS is a distributed file system. We use IPFS to host the images and metadata for the NFTs. We use the Pinata pinning service to upload the files to the network.

Why only Metamask?

This is something we put together in a few hours as an experiment. Other wallets may work, they just haven't been tested. Feel free to submit a PR on Github with additional functionality!

Is the contract audited?

No, but you can view it on etherscan. You aren't approving any tokens and your only risk is the gas cost on a failed transaction.
